# Task 4 - Form Design and Validation

This is Task 4 of my Frontend Development Internship at DecodeLabs.

## Project Title

Registration Form Validation

## Description

This project is a responsive registration form created using HTML, CSS, and JavaScript. The form validates user input and displays error or success messages based on the entered data.

The project focuses on form design, input validation, user feedback, and clean responsive layout.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- CSS Grid
- Responsive Design
- GitHub Pages

## Features

- Full name validation
- Email validation
- Phone number validation
- Course selection validation
- Password validation
- Confirm password validation
- Error messages for invalid input
- Success message after valid submission
- Responsive layout for desktop, tablet, and mobile
- Clean and professional UI design

## Project Requirement

The task was to create a form and validate user inputs using JavaScript. The form includes input fields, error and success messages, and basic validation logic.
